<a href='chatcmd:///played'>/played</a>
Tells you for how long you have played this character. Also gives you the local time, both in-game and real-time.

<a href='chatcmd:///born'>/born</a>
Tells you when character was created.

<a href='chatcmd:///stuck'>/stuck</a>
When you are stuck, sit down and use this command and you will be moved

<a href='chatcmd:///tipoftheday'>/tipoftheday</a>
Will bring up the next tip.

<a href='chatcmd:///messagebox Some text'>/messagebox Some text</a>
Opens a messagebox with [text] in it and an OK button to close it.
